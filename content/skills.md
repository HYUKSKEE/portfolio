# 기술 스택

> 실제 보유한 기술과 숙련도, 사용 경험을 구체적으로 적어주세요.

## Frontend

- **React / Next.js**: 4년 이상 사용. App Router, 서버 컴포넌트, 스트리밍 SSR 실무 경험 보유.
- **TypeScript**: 모든 프로젝트에서 strict 모드로 사용. 제네릭과 타입 유틸리티를 활용한 안전한 API 설계 경험.
- **상태 관리**: Zustand, TanStack Query를 상황에 맞게 선택하여 사용.
- **스타일링**: Tailwind CSS, CSS Modules, styled-components 경험. 디자인 시스템 구축 경험 있음.

## 협업 / 도구

- Git/GitHub 기반 협업, PR 리뷰 문화에 익숙함.
- Figma로 디자이너와 협업, 디자인 토큰을 코드로 변환하는 작업 다수 수행.
- Jira/Linear를 활용한 애자일 스프린트 경험.

## 테스트 / 품질

- Jest, React Testing Library로 단위/통합 테스트 작성.
- Playwright로 E2E 테스트 자동화 구축 경험.

## 성능 / 인프라

- Lighthouse, Web Vitals 기반 성능 모니터링 및 최적화.
- Vercel, AWS(S3/CloudFront)를 이용한 배포 경험.
- CI/CD 파이프라인(GitHub Actions) 구축 경험.

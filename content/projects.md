# 프로젝트

> 각 프로젝트마다 "무엇을, 왜, 어떻게, 결과"를 구체적으로 적으면 챗봇이 훨씬 풍부하게 답변합니다.

## 프로젝트 A — 커머스 플랫폼 리뉴얼

- **기간**: 2024.01 ~ 2024.06
- **역할**: 프론트엔드 리드 (3인 팀)
- **문제**: 기존 서비스의 첫 화면 로딩 속도가 느리고, 모바일 전환율이 낮았습니다.
- **해결**: Next.js App Router로 마이그레이션하며 서버 컴포넌트를 적극 활용, 이미지 최적화와 코드 스플리팅을 적용했습니다.
- **결과**: LCP를 3.8초에서 1.6초로 단축했고, 모바일 전환율이 22% 개선되었습니다.
- **기술 스택**: Next.js, TypeScript, Tailwind CSS, TanStack Query

## 프로젝트 B — 사내 디자인 시스템 구축

- **기간**: 2023.03 ~ 2023.09
- **역할**: 디자인 시스템 오너
- **문제**: 여러 서비스 팀이 각자 다른 스타일과 컴포넌트를 사용해 일관성이 부족했습니다.
- **해결**: Figma 디자인 토큰을 기반으로 React 컴포넌트 라이브러리를 구축하고, Storybook으로 문서화했습니다.
- **결과**: 신규 화면 개발 속도가 약 30% 향상되었고, 5개 팀이 공통으로 사용하게 되었습니다.
- **기술 스택**: React, TypeScript, Storybook, styled-components
